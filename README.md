# AgenteAsistencia

## Herramienta de IA Utilizada

Este proyecto cuenta con la asistencia de **GitHub Copilot**, impulsado por **Claude Haiku 4.5**, un modelo de IA avanzado que proporciona soporte inteligente durante el desarrollo.

### De qué manera ayuda?

- **Análisis y Exploración**: Examina la estructura del proyecto y los archivos para comprender el contexto completo.
- **Desarrollo de Características**: Ayuda en la implementación de nuevas funcionalidades y mejoras al código.
- **Debugging y Resolución de Problemas**: Identifica errores y propone soluciones efectivas.
- **Mejoras de Código**: Refactoriza y optimiza el código siguiendo mejores prácticas.
- **Documentación**: Genera documentación clara y útil para facilitar la comprensión del proyecto.
- **Automatización de Tareas**: Automatiza procesos repetitivos mediante scripts y herramientas.

## Características
- **Búsqueda en Tiempo Real**: Filtrado por nombre, categoría, fabricante o registro sanitario.
- **Sincronización con ARCSA**: Botón para descargar y actualizar el catálogo desde la fuente oficial de Ecuador.
- **Diseño Clínico**: Interfaz moderna y responsiva basada en los colores institucionales.

## Instalación

1. Clona este repositorio.
2. Instala las dependencias:
   ```bash
   pip install nicegui requests beautifulsoup4 openpyxl
   ```
3. Ejecuta la aplicación:
   ```bash
   python app.py
   ```
