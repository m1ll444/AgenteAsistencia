¿Qué es backend?
Es la lógica del programa, el cual se conecta con la base de datos para darle funcionalidad.

¿Qué es frontend?
Es la parte visual del programa, existen 3 tipos de programas: de escritorio, web y móvil.

¿Qué es una skill y dónde encontrarlas?
Una skill sirve para instalar una habilidad que puede tomar decisiones en base a lo que necesitemos.
IMPORTANTE: Debe instalarse Node.js para instalar las skills.

Consultar: firebase y supabase
Firebase: Facilita el desarollo de aplicaciones, desde la creación hasta la implementación y el monitoreo.

Supabase: Permite crear aplicaciones modernas con base de datos en tiempo real, autenticación, APIs automáticas, almacenamiento seguro y funciones en el borde (Edge Functions).
